# Gestor de Tareas FocusFlow

  

Aplicación web para la gestión de tareas basada en un tablero visual (tipo Kanban), desarrollada con **HTML5, CSS3 y JavaScript Vanilla**. Permite organizar, priorizar y hacer seguimiento de actividades de forma simple, intuitiva y responsive.

## Características

  

-  **CRUD completo de tareas:** Crear, visualizar, actualizar estado y eliminar tareas fácilmente.

-  **Tablero tipo Kanban:** Organización en columnas:

- Pendientes

- En progreso

- Completadas

-  **Sistema de prioridades:** Identificación visual mediante colores (Alta, Media, Baja).

-  **Validación de datos:**

- Título obligatorio

- Descripcion opcional

-  **Contadores dinámicos:** Número de tareas actualizado en tiempo real por columna.

-  **Diseño responsivo:** Adaptado a móvil, tablet y escritorio mediante breakpoints.

  

## Tecnologías utilizadas

  

-  **HTML5** → Estructura semántica

-  **CSS3** → Flexbox, Grid y variables CSS

-  **JavaScript (ES6+)** → Manipulación del DOM y lógica de la aplicación

## Instalación y uso

  

1. Clona este repositorio:

```bash

git clone https://github.com/xroMartin1/proyecto_tareas_gestor.git

```

  

2. Ingresa a la carpeta del proyecto:

```bash

cd proyecto_tareas_gestor

```

  

3. Abre el archivo `index.html` en tu navegador.
